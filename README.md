# ONLYOFFICE DocSpace Samples

This repository provides examples for integrating with [ONLYOFFICE DocSpace](https://api.onlyoffice.com) using **Node.js** and **Python**.

## Project Structure

- **`js-sdk/`** - Examples of working with the JavaScript SDK for DocSpace.
- **`oauth2/`** - OAuth2 authentication implementation for accessing DocSpace API.
- **`webhooks/`** - Handling events through DocSpace webhooks.
- **`api-backend/`** - Examples for building and integrating a backend API to interact with DocSpace.

## Section Overview

### 1. **JS SDK**
This section provides examples of using the JavaScript SDK to integrate with DocSpace, covering:
- File management, user management, and document/room management.

For more details, refer to the [DocSpace JavaScript SDK Documentation](https://api.onlyoffice.com/docspace/javascript-sdk/get-started/basic-concepts/).

### 2. **OAuth2**
This section shows how to set up OAuth2 authentication to access the DocSpace API securely:
- Application registration, implementing OAuth2 flow, and using OAuth2 tokens for API access.

For more details, refer to the [DocSpace OAuth2 Documentation](https://api.onlyoffice.com/docspace/oauth-api/get-started/basic-concepts/).

### 3. **Webhooks**
This section provides examples for handling webhooks in DocSpace:
- Subscribing to events such as user creation, file updates, and room structure changes.

For more details, refer to the [DocSpace Webhook Documentation](https://api.onlyoffice.com/docspace/api-backend/get-started/how-it-works/webhooks/).

### 4. **API Backend**
This section covers integrating a backend API with DocSpace:
- Setting up routes for user management, file handling, and room management using RESTful or GraphQL APIs.

For more details, refer to the [DocSpace API Backend Documentation](https://api.onlyoffice.com/docspace/api-backend/get-started/basic-concepts/).
